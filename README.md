# 五張撲克牌

此專案是在只發五張牌並判斷排型的基礎上再加上下注的功能，根據牌型獲得相應倍數的金額

使用者流程:輸入押注金額 → 按下押注按鈕 → 發牌 → 選擇要換的牌 → 換牌 → 判斷牌型 → 計算獎金

押注金額不能超過總資金
